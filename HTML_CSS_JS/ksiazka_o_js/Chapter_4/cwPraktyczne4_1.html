Ćwiczenie praktyczne 4.1

1. Utwórz zmienną z wartością boolowską.

2. Wartość tej zmiennej wyświetl w konsoli.

3. Sprawdź, czy zmienna ma wartość true. Jeżeli tak, w konsoli wyświetl komunikat za pomocą następującej składni: if(nazwaZmiennej){ // Akcja. }

4. Dodaj kolejną konstrukcję if ze znakiem ! przed nazwą zmiennej, aby w ten sposób sprawdzić, czy warunek jest nieprawdziwy. Utwórz odpowiedni komunikat wyświetlany w takiej sytuacji. Kod powinien zawierać teraz dwie konstrukcje if, w tym jedną z wykrzyknikiem. Możesz również użyć konstrukcji if z blokiem else — poeksperymentuj.

5. Zmień wartość zmiennej na przeciwną i zobacz, jak zmieni się wynik działania kodu.
