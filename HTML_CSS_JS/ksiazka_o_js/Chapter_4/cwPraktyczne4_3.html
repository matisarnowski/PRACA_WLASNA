Ćwiczenie praktyczne 4.3

  1. Utwórz wartość boolowską dla zmiennej ID.
  2. Korzystając z operatora trójargumentowego, utwórz zmienną message, której wartość będzie zależała od wartości zmiennej ID i zawierała komunikat informujący o możliwości (lub jej braku) wstępu do lokalu.
  3. Wartość zmiennej message wyświetl w konsoli.
