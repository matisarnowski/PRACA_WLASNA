Ćwiczenie praktyczne 10.3

Użyj JavaScriptu do pobierania elementów strony za pomocą nazwy znacznika.
  1. Rozpocznij od utworzenia prostego pliku HTML.
  2. Utwórz trzy elementy używające tego samego znacznika.
  3. Do każdego z tych elementów dodaj pewną treść, aby można było je odróżniać.
  4. Do pliku HTML dodaj element script, w którym umieść kod JavaScript pobierający elementy na podstawie nazw znaczników i umieszczający je w zmiennej tablicy.
  5. Za pomocą wartości indeksu pobierz środkowy element i wyświetl go w konsoli.
