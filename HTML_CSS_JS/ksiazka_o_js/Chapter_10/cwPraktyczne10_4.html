Ćwiczenie praktyczne 10.4

Pobierz wszystkie elementy strony dopasowane za pomocą nazwy klasy elementu.

  1. Rozpocznij od utworzenia prostego pliku HTML.
  2. Utwórz trzy elementy posiadające tę samą klasę. Możesz użyć odmiennych znaczników, o ile każdy z nich będzie miał tę samą klasę. Do każdego z tych elementów dodaj pewną treść, aby można było je odróżniać.
  3. Do pliku HTML dodaj element script, w którym umieść kod JavaScript pobierający elementy na podstawie nazwy klasy. Otrzymany w wyniku tej operacji egzemplarz HTMLCollection przypisz zmiennej.
  4. Indeks można wykorzystać do pobierania poszczególnych elementów HTMLCollection, podobnie jak w przypadku elementów tablicy. Począwszy od indeksu 0 pobierz jeden z elementów strony o danej nazwie klasy i wyświetl go w konsoli.
