Ćwiczenie praktyczne 10.5

Metodę querySelector() wykorzystaj do pobrania pojedynczego elementu.

  1. Rozpocznij od utworzenia prostego pliku HTML.
  2. Utwórz cztery elementy posiadające tę samą klasę. Możesz użyć odmiennych znaczników, o ile każdy z nich będzie miał klasę zdefiniowaną w atrybucie elementu.
  3. Do każdego z tych elementów dodaj pewną treść, aby można było je odróżniać.
  4. Do pliku HTML dodaj element <script>, w którym umieść wywołanie metody querySelector() pobierające pierwsze wystąpienie elementu o podanej klasie. Otrzymany w wyniku tej operacji element umieść w zmiennej. Jeżeli zostanie dopasowanych więcej elementów, metoda querySelector() zwróci jedynie pierwszy.
  5. Zwrócony element wyświetl w konsoli.
